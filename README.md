# 真央note 記事ライブラリ（ダッシュボード）

note「**真央｜更年期、はじめまして**」（40代以降・更年期世代向けの体験談note）の
全50記事を一覧・検索できる、自己完結型の閲覧用ダッシュボードです。

## ファイル

- `index.html` — これ1ファイルで完結。記事データ（タイトル・本文）はJS内にJSONとして埋め込み済み。外部依存なし。ダブルクリックでブラウザで開けます。

## 機能

- **記事一覧** — 全50本。タイトルをクリックすると本文が展開表示されます。
- **フリーワード検索** — タイトル・本文を横断して絞り込み。
- **カテゴリ絞り込み** — 以下7カテゴリを色分け表示。
  - 更年期症状・からだ / 婦人科・検診 / 相談先・オンライン診療 / パートナー・家族 / お金・費用 / セルフケア・理念 / 読者Q&A
- **ライト / ダークモード** — OSの設定に自動追従。左サイドの「ライト / ダーク切替」ボタンで手動切替も可能。
- **相談先マップ（ハブ記事）** — 集大成となる記事（30番）に「◎」バッジを表示。

## デザイン

姉妹アカウント「はる」のダッシュボード（`../dashboard/index.html`）の構造をベースに、
真央のブランドカラーである**テラコッタ（#C97B63）系**を基調とした落ち着いた配色に変更しています。

## 元データ

- 記事本文: `../mao/articles/01〜50_*.md`
- アカウントコンセプト: `../mao/account/concept.md`

`index.html` は `../mao/articles/` の全記事から自動生成しています。記事を更新した場合は再生成が必要です。
